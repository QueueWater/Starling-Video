Classpublic class VideoEvent
InheritanceVideoEvent starling.events.Event

A VideoEvent is triggered once per video/camera frame and dispatched by Video/WebcamVideo. They may come in handy if you want to handle the draw/upload process yourself; if you want to process a BitmapData/ByteArrays only if it gets updated (AR); or if you want to use a custom renderer for the video.
